Preguntas frecuentes

Lo que probablemente te estás preguntando.

¿Esto es legal? ¿Puedo consultar a alguien sin que sepa?

Sí, es legal cuando la consulta tiene base legítima de negocio (contratación, crédito, alianza). Para consultas que tocan datos sensibles como Buró, debes recabar consentimiento ARCO firmado por el evaluado. TrustScore te entrega plantillas de aviso de privacidad y consentimiento listas para usar.

¿Necesito ser empresa o puedo usarlo como persona física?

Ambos. Puedes pagar como persona física con tarjeta o como empresa con factura (CFDI 4.0 emitida por nosotros). Algunos tiers requieren validación de cuenta para evitar uso indebido.

¿Qué tan exacto es el score?

El score 0–1000 combina señales de 8 fuentes oficiales con pesos calibrados según severidad (estar en 69-B pesa más que tener una queja menor en CONDUSEF). Reporte siempre incluye el detalle de qué señales lo movieron, así puedes auditar y discutir.

¿Tienen API o solo dashboard?

Ambos. El dashboard es para consultas ad-hoc. La API REST es para integración con tus sistemas, marketplaces o flujos de aprobación. Documentación tipo Stripe, SDKs en Python y JavaScript próximamente.

¿Qué pasa si el reporte falla o falta data?

Si una fuente devuelve error o no tiene info de la persona, el reporte lo indica explícitamente y no te cobramos esa fuente. Cero "data inventada" o relleno con probabilidades.

¿Cómo se compara con Truora, MetaMap, Mati?

Esos están enfocados en KYC para fintechs (validar que un usuario nuevo es quien dice ser). TrustScore está enfocado en evaluación reputacional B2B: ¿le hago negocio a este wey? Diferente caso de uso, diferente data, diferente score.
